## ğŸ› ï¸ Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- Yarn or npm
- Expo CLI:
```npm install -g expo-cli```

### Installation for development

```
git clone https://github.com/AbdourahmaneGadio/MovieDLE.git
cd MovieDLE
npm install
```

### Start the App
```
npx expo start
```
